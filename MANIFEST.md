## Known, disclosed gaps (not oversights — intentionally scoped out and documented)
- Online's clue cards don't visually match hot-seat's card styling yet (behavior is identical).
- Return/Frustration use a fixed optimal-happiness power (102) rather than a real friendship-based formula — no friendship stat exists in this draft context.
- Jump Kick/High Jump Kick's crash damage (1/8 max HP) is an estimate — no more Gen-2-specific figure found with higher confidence.
- Hidden Power is a fixed Normal-type 60-BP move. Its real per-Pokémon variable type/power can't be derived (no DVs/IVs here). The ideal "Hidden Power (X)" random-per-draft type is a cross-cutting draft-layer + UI feature, deferred as its own task (out of the sim-only scope).
- Weather: Solar Beam's power reduction in rain/sandstorm is not modeled (only its instant-charge in sun is); there are no weather-extending held items, so all weather lasts a fixed 5 turns.
- Substitute: drain/recoil use the move's computed damage rather than the amount the sub actually absorbed (minor over-heal/recoil when overkilling a sub); Pain Split ignores the sub and averages real HP; a confusion self-hit strikes the mon, not its own sub.
- The move audit (all 244 Gen-2 moves) is complete: every draftable move now behaves per Gen 2 within the switchless-1v1 scope, save the approximations listed here. Moves that are meaningless in a 1v1 with no switching/items/PP (switch-forcing, entry hazards, item theft, PP manipulation, gender, party-wide, dynamic-moveset copies) are banned from the draft pool rather than modeled — a deliberate scope boundary, listed in draft.js's BANNED_DRAFT_MOVES.
- Silhouette sprite assets (docs/img/silhouettes/<num>.png) are user-supplied and optional. Naming is by National Dex number as a plain integer string (1.png … 251.png, no zero-padding), matching gen2.json's `num` field. **As of 2026-07-14 the silhouette `<img>` IS wired** — it renders on exactly two screens (the Single-Player post-game reveal card and the Pokédex detail view), both via the shared pokeinfo.js builder. The files are shown as-is with the `.poke-silhouette` class (no filter — they're expected to already be silhouettes, per the user's legal-caution note for a live site), and a missing file hides itself gracefully. Expected filenames + behaviour documented in docs/img/silhouettes/README.md. NOTE: the separate `.draft-silhouette` CSS (72×72, brightness(0) invert(.15) blackout filter) and the `silhouetteSpecies`/`silhouetteSpriteId` data on the draft card remain UNWIRED — draft cards still have no `<img>`; the reveal/Pokédex silhouettes are a distinct, complete feature.
- Background-audio assets are user-supplied and optional — the music system (lib/music.js 2.0.0, main.js 1.6.0) is fully wired but inert until files exist. **Music** (docs/audio/music/*.mp3): every mode now has its OWN dedicated file — menu.mp3, draft-battle.mp3, daily-challenge.mp3, guess.mp3, safari.mp3, victory-road.mp3, multiplayer.mp3, online.mp3, cycling-road.mp3, pokedex.mp3, leaderboard.mp3 — plus default.mp3, the universal fallback for any mode whose own file is missing/404s or that isn't in the map at all (a real runtime check, not just a lookup default). **SFX** (docs/audio/sfx/*.mp3): enter-guess.mp3 (single/safari/victoryroad/hotseat/online), to-menu.mp3 (returning to the main menu), transition.mp3 (every other destination, AND the fallback if enter-guess.mp3/to-menu.mp3 is missing), game-start.mp3 (a round actually starting — wired into all 7 modes that have a real "start" action; falls back to transition.mp3 if not yet supplied). Filenames + the "start" trigger points documented in docs/audio/music/README.md and docs/audio/sfx/README.md. Missing files fail silently or fall back per the above. Reminder: use original/royalty-free audio, not clips from the official games (live site).
